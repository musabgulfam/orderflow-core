# orderflow-core

This project is the backend of an online order automation system designed for on-ground bookers of a candy manufacturer. Bookers are individuals who maintain relationships with retailers and shops, and convey orders to sub-distributors. The system streamlines the process of collecting, managing, and forwarding orders, making it easier for bookers to operate efficiently and for manufacturers to track order flow.

## Features
- Order collection and management
- Booker and retailer relationship tracking
- Automated order forwarding to sub-distributors

## Getting Started
To run the project:

```bash
go run main.go
```

## Requirements
- Go 1.18 or higher
